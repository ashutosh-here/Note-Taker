# Note Taker Webapp

A simple web application for note taking  using MySql database and *JAVA* as back-end , applying ORM principles of <b>Hibernate Framework</b>.

## Tools & Technologies used 
### *Front-end*

* HTML5
* CSS3
* Javascript (ES6)
* JQuery
* Bootstrap 4
* AJAX

### *Back-end* 

* Servlet
* Java Server Page (JSP)
* MySql
* Maven build tool

## *Framework*
* Hibernate 5.2
